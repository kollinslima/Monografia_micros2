\chapter{Conclusão}
\label{Conclusao}

\par Neste trabalho foi apresentado uma proposta para realizar o reconhecimento de sinais, aplicada à Língua Brasileira de Sinais - LIBRAS.
\par Pode-se dizer que o trabalho atinge parcialmente seus objetivos. Por um lado, foi implementado um sistema capaz de ler, processar, classificar e responder textualmente os símbolos apresentados por meio da câmera de um \textit{smartphone} Android. Por outro, com uma taxa de erro superior à 60\%, o sistema não é capaz de fazer a classificação correta para todos os símbolos numéricos propostos.

\par Apesar dos resultados obtidos, o desenvolvimento deste projeto exigiu conhecimento de diversas áreas, o que foi de grande valia para o aprendizado. Entre os conhecimentos obtidos, pode-se destacar as diversas técnicas de processamento de imagem que foram necessárias na etapa de pré-processamento (com auxílio da biblioteca \textit{OpenCV}); o uso da Raspberry-pi como elemento central no projeto; o uso do \textit{Flask} para desenvolver a interface \textit{web}; o uso do banco de dados \textit{MongoDB} além da prática em desenvolvimento Python e Android.

\par Por fim, analisando o que foi desenvolvido, notou-se que esta proposta não é suficiente para prover independência ao indivíduo surdo, que o objetivo principal. Isso se deve ao fato de o sistema não ser de fácil uso no dia-a-dia, mesmo se estivesse em perfeito funcionamento, uma vez que depende de uma câmera apontando para o usuário, fazendo com que o aplicativo dependesse de duas pessoas para seu uso (uma segurando a câmera e outra fazendo os gestos), do contrário restringiria os movimentos por exigir que uma das mãos segure a câmera. Seu uso seria mais eficiente em um contexto de palestra ou algum outro evento onde já existe uma câmera externa monitorando o ambiente.

\par Para trabalhos futuros, é preciso pensar em outras maneiras de fazer o reconhecimento de gestos, que seja de fácil uso por uma única pessoa e que não envolva uma quantidade excessiva de hardware externo, tanto para prover um sistema mais acessível, quanto devido à estética.
